# Drishti Shekhawat Portfolio

Static single-page portfolio for Drishti Shekhawat.

## Files

- `index.html` - page markup and content
- `styles.css` - responsive layout, animated background, and section styling
- `script.js` - particles, reveal animation, active nav, and mobile menu

## Photo

Add the profile photo at:

```text
assets/drishti-profile.jpg
```

If the image is missing, the page shows a clean `DS` fallback in the hero portrait area.
